## 🎨 Interface do Sistema

Desenvolvido apenas a interface do sistema utilizando:

- HTML  
- CSS  
- JavaScript  

---

## 🧩 Funcionalidades da Interface

- Cadastro de requisitos com validação de campos  
- Campos do requisito:
  - Título  
  - Descrição  
  - Classificação (funcional / não funcional)  
  - Prioridade (baixa, média, alta)  
  - Status  

- Armazenamento local usando `localStorage`  
- Listagem de requisitos cadastrados  
- Filtros por tipo e status  
- Exibição de métricas básicas:
  - Total de requisitos  
  - Funcionais  
  - Não funcionais  
- Exclusão de requisitos
